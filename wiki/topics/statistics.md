# Statistics & Probability

## Distributions
- **Normal**: μ, σ² — CLT, z-tests
- **Binomial**: n, p — count of successes
- **Poisson**: λ — rare events, count data
- **Exponential**: λ — time between events

## Hypothesis Testing
- **z-test**: known σ, large n
- **t-test**: unknown σ, small n
- **chi-squared**: categorical data, goodness of fit
- **Mann-Whitney U**: non-parametric alternative to t-test

## Confidence Intervals
- CI = x̄ ± Z_α/2 × (σ/√n)
- Interpretation: if we repeat the experiment many times, 95% of CIs will contain the true mean

## Bayesian vs Frequentist
- **Frequentist**: fixed parameter, random data → p-values, CIs
- **Bayesian**: random parameter, fixed data → posterior, credible intervals
- Bayesian advantage: natural interpretation, prior knowledge, no multiple testing penalty

## Key Concepts
- **Central Limit Theorem**: sample means → normal as n → ∞
- **Law of Large Numbers**: sample mean → population mean
- **Type I error (α)**: false positive — reject H₀ when true
- **Type II error (β)**: false negative — fail to reject H₀ when false
